# StartupForge Client

React + Vite client with role-based dashboards, Framer Motion, Recharts, TanStack Query, and Better Auth.

## Run

```bash
cp .env.example .env
npm install
npm run dev
```

## Verify

```bash
npm run lint
npm run build
```

Read the root `MASTER-SETUP-GUIDE.md` for all environment and deployment instructions.
